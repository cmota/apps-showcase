# Matchday 26 — marketing site

Static landing page + privacy policy for **Matchday 26**, an independent, unofficial
companion app for the 2026 World Cup (Android · `com.cmota.matchday`).

- **Live site:** https://cmota.github.io/matchday26/
- **Privacy policy:** https://cmota.github.io/matchday26/privacy.html

Plain HTML/CSS, no build step. Hosted on GitHub Pages from the `main` branch root.
